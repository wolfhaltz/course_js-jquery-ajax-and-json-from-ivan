De acordo com a condição do 'if', definimos se aquilo vai ser executado ou não.

Se houver outras condições, que são os 'else if', ele vai percorrer. Do contrário, ele vai ir direto para o else. 

                    var age = 17
                    if(age < 18){
                        console.log('-18')
                    }else if(age > 18){
                        console.log('+18')
                    }else{
                        console.log('what are you doing?????')
                    }

Exemplo prático

Se, um aluno, tiver a média igual ou maior à 7, e, suas faltas forem iguais ou inferiores a 4, ele está aprovado.

Solução:

                    var grades = 8;
                    var absences = 4;
                    
                    if(grades >= 7 && absences <= 4){
                        console.log('Approved!')
                    }else{
                        console.log('Disapproved!')
                    }
