Loops são estruturas de repetição de comandos, então se eu quiser repetir algo, escrevo uma vez e defino quantas vezes ele vai ser repetido

A primeira expressão é alguma coisa que vai ser executada uma vez antes do loop começar, ou seja, inicializamos uma variável. O segundo comando é uma condição para que o loop seja passado. Já o terceiro comando que vamos executar em todas as passagens do loop, na final da passagem, ou seja, incrementando o valor da primeira expressão para que o loop passe

No exemplo a baixo, o loop passará 5 vezes:

for(var a = 0; a < 5; a++){ console.log(a) }

Agora, como podemos percorrer um array usando um for?

var students = ['Peter', 'Marie', 'Joseph'] for(var b = 0; b < students.length; b++){ console.log(students[b]) }

Note que usamos 'b' como índice para percorrer cada aluno
