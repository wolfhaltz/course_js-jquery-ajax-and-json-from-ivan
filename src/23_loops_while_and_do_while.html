O loop while, é como um for, mas dentro dos parênteses, colocamos somente a condição

                        var count = 0;
                        while(count < 5){
                            console.log(count);
                            count++;
                        }

Já no loop do/while, ele vai repetir o código enquanto houver a condição.

Então, primeiro ele executa o código e, depois ele confere se a condição é válida, de modo que ele sempre vai rodar uma vez, como no exemplo abaixo:

                        var count2 = 10;
                        do{
                            console.log(count2);
                            count++;
                        }while(count2 < 5);
