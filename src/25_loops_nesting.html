Nesting é quando há um condicional dentro do outro.

Exemplo 01

Ao receber os dados de um formulário, recebemos:

  • Idade
  • Se é ou não sócia

Baseado nisso, temos 4 faixas de preço:

  • Sócios: grátis
  • Não sócios menores de 18: R$ 6,00
  • Não sócios maiores de 18: R$ 12,00
  • Maiores de 65 anos: grátis

Solução:

                    var partner = false
                    var age = 25
                    
                    if(partner == true || age >= 65){
                        console.log('Free')
                    }else{
                        if(age >= 18){
                            console.log('R$ 12,00')
                        }else{
                            console.log('R$ 6,00')
                        }
                    }
